## Project Overview

A western Massachusetts community organization has requested a map of locations where ICE (Immigration and Customs Enforcement) has 287g contracts.  This project will develop an interactive map displaying this information, and pairing it with other relevant community resources.

## The Problem

Local community organizations have asked for a way to communicate - in clear, easy to understand, and linguistically accessible ways - where communities might be vulnerable to ICE agents.  Local groups have asked for a dashboard that:
 - displays updated information about 287g contracts
 - displays information in translation from English (especially in Spanish, and Mandarin)
 - provides contact information for local law enforcement 

## Our Solution

We're developing a dashboard that includes:

#### Up-to-date Information
Using data from authorized and official government sources, we display the very latest data. 

#### Multilingual Documentation
Plain-language documentation in multiple languages that explains the contracts in each locale.

#### Modular Components
A modular development gives us the ability to adapt this prototype to a subsequent co-design process.


## Technical Approach

The dashboard is built using:

- **Python** for data processing and analysis, and _streamlit_ for easy display
- **GitHub Actions** for automated testing and deployment

## Partner Organizations

By the end of the first phase of hte project, DIG will have built:
